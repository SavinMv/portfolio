# Savin MV — Portfolio

Personal portfolio website built with **React.js + Vite**, deployed on **Vercel**.

🌐 **Live:** [savin-portfolio.vercel.app](https://savin-portfolio.vercel.app)

## Tech Stack

- React.js (Vite)
- Inter + Space Mono (Google Fonts)
- Pure CSS-in-JS (no external UI library)
- Deployed on Vercel

## Sections

- **Home** — Typewriter intro, stats, CTA
- **About** — Bio, skills by category
- **Projects** — FinTrack AI, JobSphere
- **Experience** — SuprMentr Technologies internship + certifications
- **Contact** — Email, LinkedIn, GitHub

## Run Locally

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Push to GitHub → Import on [vercel.com](https://vercel.com) → Auto deployed ✅
